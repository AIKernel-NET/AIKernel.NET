## Description

This technical note is Part I-1 of the AIKernel / AIOS Phase-1 Specification Series. It defines the ROM Format and Knowledge Snapshot Model as the invariant knowledge substrate for deterministic AIOS governance. ROM, or Relation-Oriented Markdown, is treated as a governable unit of knowledge from which canonical identity, provenance, relation topology, signature verification, and replayable context reconstruction can be derived.

The English manuscript is the canonical version. The Japanese manuscript is included as a companion translation. Both Markdown manuscripts use the AIKernel standard YAML header for consistent management across the Phase-1 and Phase-2 paper series.
