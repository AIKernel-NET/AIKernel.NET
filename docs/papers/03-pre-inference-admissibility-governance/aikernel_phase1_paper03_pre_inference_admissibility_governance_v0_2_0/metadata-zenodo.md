## Description

This technical note is Part II-3 of the AIKernel / AIOS Phase-1 specification series. It defines Pre-Inference Admissibility Governance as the deterministic admission-control layer that determines whether an inference request may enter the AIKernel execution pipeline.

The paper introduces prompt-injection and override gating, capability admission, critical-operation control, computational-complexity gating, fail-closed requirements, and replayable admission evidence. It connects Paper 01's governed ROM knowledge substrate and Paper 02's VFS capability model to the governance layer that controls whether stochastic inference may begin.

The English manuscript is the canonical version. The Japanese manuscript is included as a companion translation.
