## Description

This bilingual technical report proposes Interface-Led Architecture (ILA), a software development methodology for the age of AI-assisted implementation. ILA builds software from interfaces, contracts, and responsibility boundaries rather than from implementation-first code, and positions AIKernel as a validation environment for deterministic execution boundaries, ROM-based design knowledge, Provider separation, and ReplayLog-based reproducibility. The package includes both Japanese and reviewed English manuscripts.
